Select Filter

Overviews

The select filter will fetch its data by calling the fetchData function defined for the column. The fetchData function must return either:

  • an array
  • a function that returns an array or a promise that resolves to an array
  • a promise that resolves to an array
  • a promise that resolves to a function that returns an array or a promise that resolves to an array
